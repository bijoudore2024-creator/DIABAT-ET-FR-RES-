Synchronisation multi-utilisateur (Supabase)

Connecte cette pharmacie à un projet Supabase pour activer la sauvegarde en ligne, la synchronisation temps réel entre appareils et la connexion multi-utilisateur. Sans configuration, l'application continue de fonctionner normalement en local sur cet appareil.

Comment obtenir ces informations ?
  1. Crée un projet gratuit sur supabase.com
  2. Dans le Dashboard : Project Settings > API, copie la "Project URL" et la clé "anon public"
  3. Dans le SQL Editor, exécute le script de création de table (voir documentation du projet)
  4. Dans Authentication > Providers, active "Email"
  5. Colle les deux valeurs ci-dessus puis clique sur "Enregistrer et se connecter"

Assistant IA locale (LM Studio)

Connecte l'application à un serveur LM Studio (local ou sur un autre ordinateur du réseau) pour activer l'assistant IA. L'assistant peut répondre à des questions générales et consulter le stock, les ventes et les créances clients pour donner des réponses précises.

Comment configurer LM Studio ?
  1. Installe et ouvre LM Studio sur un ordinateur (télécharge un modèle, ex : Llama 3.1 8B Instruct)
  2. Dans l'onglet Local Server, clique sur "Start Server" (par défaut sur le port 1234)
  3. Si l'application est ouverte sur le même ordinateur que LM Studio, utilise http://localhost:1234/v1
  4. Si l'application est ouverte sur un autre appareil du réseau (téléphone, autre PC), utilise l'adresse IP locale de l'ordinateur qui fait tourner LM Studio, ex : http://192.168.1.10:1234/v1 — assure-toi que le pare-feu autorise les connexions entrantes sur ce port
  5. Renseigne le nom exact du modèle chargé (visible en haut de LM Studio), puis clique sur "Tester la connexion"

⚠ Zone de danger

Supprime définitivement toutes les données de test (produits, stock, ventes, clients, ordonnances, fournisseurs, commandes, gardes, finances, mouvements) pour repartir avec une base vide, prête pour la production. Les paramètres de la pharmacie (nom, logo, seuils, connexion Supabase) et les comptes utilisateurs sont conservés.

Cette action est irréversible et ne peut pas être annulée.
